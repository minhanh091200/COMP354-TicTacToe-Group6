Getting Started

To get a local copy up and running follow these simple steps.

Prerequisites

There are no special requirements to play this game apart from having a Web Browser that supports HTML5 and CSS3.

Setting up the Project

  1. Clone the repo.
    2. git clone https://github.com/yelynn1/tictactoe.git
  2. Open the index.html file in any web browser.
